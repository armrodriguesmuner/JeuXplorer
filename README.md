# LienSocial

Application mobile (Android & iOS) de lien social gamifiée basée sur des interactions physiques réelles.

## Concept

Les utilisateurs gagnent des points en scannant des QR codes placés dans la ville et en validant des rencontres réelles avec d'autres Agents (identité anonyme "Agent XXX"). L'application encourage les liens humains authentiques sans jamais ressembler à une application de rencontre amoureuse.

## Structure du projet

```
agents/          — Définitions des agents IA du projet
  discovery/     — Agent Méthode & Besoin (phase de formalisation du besoin)
  _base/         — Ressources partagées entre agents
livrables/       — Documents produits par les agents
  01_discovery/  — Livrables de la phase Discovery
  besoin/        — Note de cadrage, CDCF, user stories, critères d'acceptation
memory/          — Contexte projet et décisions partagés entre agents
```
